# WizardingWorld
캐주얼 RPG 게임




## 프로젝트 소개
- PalWorld 게임을 모티브로 만든 게임.
- 각종 아이템과 몬스터를 이용해 전투에 활용할 수 있습니다. 




## 개발 환경
- 개인 프로젝트 
- 엔진 : Unity
- 버전 및 이슈관리 : Github
